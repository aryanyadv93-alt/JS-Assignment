# JS-Assignment
## Question 1 – Digit Gatekeeper
**Logic:**
Check numbers in a given range, select those divisible by K, ignore numbers containing 0, and count those whose digit sum is prime.
**Approach:**
Used loop to go through range, string conversion for digit check, and a function to check prime numbers.
**Complexity:**
Time: O(n)
Space: O(1)
## Question 2 – Roll-Seed Lock
**Logic:**
Apply 3 operations on a number based on even/odd condition, then check if result is a 3-digit number and its middle digit matches seed.
**Approach:**
Used loop for 3 steps and conditions for even/odd. Extracted middle digit using math operations.
**Complexity:**
Time: O(1)
Space: O(1)
